# 100 Days of DSA

This repository contains my solutions for a 100 Days of Data Structures and Algorithms (DSA) challenge.

## About the Project
- The challenge spans 100 days.
- Each day includes 2 coding questions (`Day X Q1.c` and `Day X Q2.c`).
- All solutions are written in **C**.
- The problems are primarily sourced from popular coding platforms like **LeetCode** and **GeeksforGeeks**.
- This repository tracks my daily progress and problem-solving journey throughout the challenge.

## Structure
The files are named consistently for easy navigation:
- `Day 1 Q1.c`
- `Day 1 Q2.c`
- ... up to `Day 100 Q2.c`

## How to Compile and Run
To test any of the solutions locally, you can use a standard C compiler (like GCC):
```bash
gcc "Day 1 Q1.c" -o solution
./solution
```
